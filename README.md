# Clarity - Control de Gastos Inteligente 💸✨

Una aplicación web de alta fidelidad para el control y análisis de tus finanzas personales, construida con una interfaz moderna y estética oscura de estilo glassmorphism.

---

## 🌟 Características Clave

- **Estética Glassmórfica Premium**: Interfaz moderna con fondos translúcidos, bordes brillantes y sombras profundas, optimizada con una tipografía pulida y un esquema de color armonioso.
- **Gráficos Interactivos (Chart.js)**: Visualiza tu distribución de gastos por categoría y analiza tu historial y tendencias mensuales al instante con gráficos interactivos y dinámicos.
- **Límites de Presupuesto con Alertas de Neón**: Define presupuestos mensuales por categoría. El sistema te alertará visualmente cambiando el color del indicador (verde, amarillo o rojo brillante) según tu consumo.
- **Búsqueda, Filtros y Ordenación**: Busca y filtra transacciones rápidamente por descripción, tipo, categoría y método de pago. Ordena columnas en tiempo real.
- **Exportación e Importación de Datos**: Realiza copias de seguridad de tus datos descargando un archivo JSON o exporta tus transacciones a un archivo CSV compatible con hojas de cálculo.
- **Efectos de Sonido Interactivos**: Cuenta con un sintetizador web nativo integrado que emite sutiles notificaciones auditivas al guardar, eliminar o alertar presupuestos.
- **Soporte PWA Desconectado (Capacidad Offline)**: Registrada como Progressive Web App. Funciona offline y se puede añadir como aplicación nativa en la pantalla de inicio de tu celular o computadora.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5 Semántico**: Estructura limpia y accesible de una sola página (SPA).
- **CSS3 Moderno**: Diseño responsivo y adaptable mediante Flexbox/Grid, variables CSS, micro-interacciones de estados hover, filtros de desenfoque y animaciones fluidas.
- **JavaScript Vanilla**: Lógica de negocio auto-contenida con persistencia inmediata en el almacenamiento local del navegador (`localStorage`).
- **Chart.js**: Biblioteca ligera de gráficos renderizados en lienzo HTML5.
- **Lucide Icons**: Set de iconos modernos y consistentes de trazo limpio.

---

## 🚀 Cómo Empezar a Usar

1. **Abrir la Aplicación**: Simplemente abre `index.html` en cualquier navegador moderno.
2. **Cargar Datos de Demostración**: Si deseas ver cómo luce la aplicación inmediatamente con información de prueba, haz clic en el botón **"Cargar Demo"** del panel superior.
3. **Guardar en tu Teléfono (PWA)**:
   - En **iOS (Safari)**: Presiona el botón de compartir y selecciona *"Agregar a la pantalla de inicio"*.
   - En **Android (Chrome)**: Presiona los tres puntos superiores y selecciona *"Instalar aplicación"*.
